Option 1 — SVG → Canvas with Isometric Matrix Transform

The SVG is parsed in JavaScript, each <path> is drawn onto a Canvas with a custom isometric 2D transform matrix. CSS transforms (skew, rotate) can then be applied to the canvas element without distorting the underlying geometry.

Canvas — Isometric Matrix Transform

option 1
Each SVG <path> is drawn onto a <canvas> element with ctx.setTransform(a, b, c, d, tx, ty) applying the isometric matrix. The canvas can then receive arbitrary CSS transforms without distorting the projected geometry.
